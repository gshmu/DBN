Before you begin
(unless already provisioned by your company)

You may need to:
  • request privileges to execute DBMS_CLOUD and DBMS_CLOUD_AI packages
  • provide api credentials for one of the supported AI Providers (OpenAI, Cohere or OCI)
  • configure AI profiles exposing certain parts of your data model

NOTE
This is an early EXPERIMENTAL feature. Please use with caution and report the issues you encounter to the DBN plugin developers.
